# Roadmap

## Phase 1: Foundation (Current)
- Core schema, adapters, ingestion, DQ, research skeleton, T212 read-only

## Phase 2: Data Enrichment
- Exchange calendar population
- Macro data pipeline (FRED, BEA, BLS, Treasury)
- OpenFIGI bulk enrichment
- Cross-source price reconciliation
- Advanced DQ rules

## Phase 3: Research Expansion
- Factor models (value, momentum, quality, size)
- Stock screeners
- Portfolio analytics
- Backtesting framework (walk-forward + OOS + bootstrap)

## Phase 4: Controlled Execution
- Trade 212 demo trading
- Position sizing algorithms
- Execution quality monitoring
- Risk management framework

## Phase 5: Production Hardening
- Airflow/Prefect migration
- Monitoring and alerting
- Performance optimization
- Live trading enablement (with safeguards)
