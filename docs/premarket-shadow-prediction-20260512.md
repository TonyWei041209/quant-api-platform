## 4. Ruleset (deterministic, conservative)

| Signal | Definition | Range |
|---|---|---|
| `momentum_signal` | `+1` if `change_1d_pct ≥ +2.0`, `-1` if `≤ -2.0`, else `0` | -1..+1 |
| `extension_signal` | `-1` if `week52_position_pct ≥ 95` AND `change_1d_pct ≥ 8` (extreme extension → mean-reversion bias), else `0` | -1..0 |
| `news_signal` | `+1` if `recent_news_count ≥ 3` AND `research_priority ≥ 4` (news only counts when corroborated by other priority signals), else `0` | 0..+1 |
| `composite` | sum of the three above | -3..+3 |

**Direction**

| composite | direction |
|---|---|
| `≥ +1` | `up` |
| `0` | `flat` |
| `≤ -1` | `down` |

**Bucket** — shadow v1 NEVER predicts the extreme buckets
(`above_plus_3` / `below_minus_3`):

| composite + dq | bucket |
|---|---|
| any with `data_quality=weak` | `minus_1_to_plus_1` |
| direction=up, extension=-1 | `minus_1_to_plus_1` (extension dampens) |
| direction=up, composite ≥ +2 | `plus_1_to_plus_3` |
| direction=up, composite=+1 | `minus_1_to_plus_1` |
| direction=down, composite ≤ -2 | `minus_3_to_minus_1` |
| direction=down, composite=-1 | `minus_1_to_plus_1` |
| direction=flat | `minus_1_to_plus_1` |

**Confidence** — capped at `medium` for shadow v1:

| condition | confidence |
|---|---|
| `data_quality` ∈ {weak, partial} | `low` |
| `|composite| ≥ 2` AND `signal_strength=high` AND no `risk_flags` | `medium` |
| everything else | `low` |

**Why this is conservative on purpose**: the universe today is heavy
with names at the 52-week high after large single-day moves
(MU +15.49%, AMD +11.44%, INTC +13.96% on `2026-05-08`). The
extension dampener neutralizes the momentum on those names — that's
the rule's pre-registered position. If they keep ripping up, the
rule was wrong; if they stall or mean-revert, the rule was right.
**No cherry-picking after the fact.**
